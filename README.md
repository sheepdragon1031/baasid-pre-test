# Getting Started with Create React App

這個項目是用 [Create React App](https://github.com/facebook/create-react-app).

## 如何使用

在開發模式下運行應用程序。打開 http://localhost:3000 在瀏覽器中查看。

```=
$ yarn install & yarn start
```
